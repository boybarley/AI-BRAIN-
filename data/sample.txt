Tentang AI Brain

AI Brain adalah platform asisten AI berbasis RAG (Retrieval-Augmented Generation)
yang dapat menjawab pertanyaan berdasarkan dokumen dan konten website Anda.

Fitur Utama:
1. Upload dokumen (PDF, TXT, DOCX, CSV, XLSX) sebagai knowledge base
2. Crawl website otomatis untuk mengambil informasi
3. Integrasi multi-channel: REST API, Telegram, WhatsApp, Messenger
4. Mendukung berbagai LLM provider: OpenRouter, OpenAI, Groq, Ollama
5. Pencarian semantik menggunakan FAISS vector database

Cara Penggunaan:
- Taruh dokumen Anda di folder data/
- Jalankan: python ingest_data.py
- Tanyakan via API: POST /api/v1/query {"query": "pertanyaan Anda"}
